# SocialSparkApp
## Description
Social Spark App is an Android application used to encourage users to maintain social connections by suggesting social actions based on the time of the day. The app was developed in Android Studio using Kotlin. 

## Features
* Accepts the user input for time of day ( morning, afternoon, dinner etc.)
* Suggets something you can say at that specific time of day.
* Includes a reset button to clear input and output
* Simple user-friendly interface

  ## Technologies Used
  * Kotlin
  * Android Studio
  * GitHub
  * GitHub Actions
 
    ## How It Works
    The app uses if statements to check the users input and display a relevant suggestion.
    Eg. Morning > Send a "Good Morning" message

    ## GitHub Actions
    Automated build is set up using GitHub Actions to ensure the app builds successfully.
